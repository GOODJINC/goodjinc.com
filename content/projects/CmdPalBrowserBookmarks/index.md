Browser Bookmarks for Command Palette는 Microsoft PowerToys Command Palette에서 브라우저 북마크를 바로 검색하고 열 수 있게 해주는 확장 프로그램입니다. Edge, Chrome, Firefox에 저장된 즐겨찾기를 자동으로 읽어오므로 Command Palette에 같은 북마크를 다시 등록할 필요가 없습니다.

이 확장은 브라우저별 프로필 선택, 여러 프로필 동시 검색, 한글 초성 검색, 영어/한국어 UI, 북마크 출처 브라우저로 열기, URL/제목/Markdown 링크 복사 같은 기능을 제공합니다. 북마크 데이터는 사용자 PC에서 로컬로만 처리되며 외부 서버로 전송하지 않습니다.

현재 GitHub Release로 배포하고 있으며, Microsoft Store와 WinGet 등록도 준비하고 있습니다.
