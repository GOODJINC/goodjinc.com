약 10개월 동안 [투두이스트(Todoist)](https://www.todoist.com/)와 [틱틱(TickTick)](https://ticktick.com/)을 모두 유료로 사용하면서, 두 앱을 실사용 관점에서 비교해보았습니다.

사실 Todoist는 수년간 사용해왔지만, 작년에 구글 캘린더와의 양방향 동기화를 없애면서 갈아타기 위해 두 앱을 비교분석하기 시작했습니다.

물론 둘 다 충분히 완성도 높은 투두 앱이지만, 사용 패턴에 따라 장단점이 극명하게 갈립니다.

그래서 이 글에서는 기능 나열이 아닌, 실제로 쓰면서 체감한 차이점을 중심으로 정리해보려고 합니다.

---

## Todoist

### 장점

- **깔끔하고 직관적인 디자인**\
  전반적으로 여백과 색 사용이 안정적이고, 처음 써도 구조나 구성을 이해하기 쉽습니다.

- **한글 지원 완성도 높음**\
  한글 입력, 검색, 표시 모두에서 불편함이 거의 없습니다.

- **자연어 입력이 매우 강력함**\
  예시. `오늘부터 내일까지 XXX 하기` 같은 문장을 입력하면 날짜·기간을 정확히 인식합니다.
  하지만 TickTick은 이런 자연어 처리가 불가능하거나 어렵기 때문에 자연어 처리는 Todoist가 압승이라고 생각됩니다.
  {{< figure src="img/nlp-in-todoist.png" alt="Todoist의 자연스러운 자연어 처리" >}}

---

### 단점

- **캘린더 뷰가 매우 부족함 (가장 큰 단점)**
  - Google Calendar 일정과 함께 보기 불가
  - 필터를 설정하여 여러 프로젝트를 한 캘린더에서 봐도 `색상 구분`이 없어 프로젝트 구분이 어려움
  - 목록과 캘린더 보기만 바꿔가면서 보고싶은데 (목록/보드/캘린더 순서로 전환해야 하기 때문에 불필요한 구성이 중간에 끼여있음)
  - 주간/월간 캘린더 보기 전환 단축키 없음

- **Google Calendar 양방향 동기화 미지원**\
  과거에는 가능했으나 현재는 Todoist → Google Calendar 단방향 보기만 가능하며, 데이터 주권은 Todoist에 있음

- **상대적으로 비싼 가격**\
  기능 대비 가격 인상 체감이 큽니다. 특히 Google Calendar와의 양방향 동기화가 빠진 상태에서 가격이 오른 점은 아쉽습니다.

- **단축키 커스터마이징 불가**\
  고정된 단축키 체계이며, 개인 작업 흐름에 맞게 조정할 수 없습니다.

- **Windows 앱 UI 버그**\
  PC 부팅 후 실행 시 창이 화면 밖으로 나가 있는 경우가 반복적으로 발생합니다.\
  제가 [프로그램 창이 화면 밖으로 사라졌을 때 해결하는 3가지 방법](/posts/recover-off-screen-window-windows-11/)이라는 게시글을 쓴 이유가 Todoist 때문입니다.

- **완료된 작업 검색이 불편함**\
  기본 검색에서는 완료된 작업이 제외되며, 별도 버튼(완료한 작업 검색하기)을 눌러야 확인 가능합니다.
  {{< figure src="img/inconvenient-search-for-completed-tasks.png" alt="Todoist에서 완료된 작업 검색 불편" >}}

- **피드백 반응이 느림**\
  버그 리포트나 기능 개선 요청에 대한 응답이 거의 없거나 매우 늦습니다.

## TickTick

### 장점

- **Google Calendar 완전한 2-way sync 지원**\
  TickTick과 Google Calendar 어느 쪽에서 수정해도 양쪽에 즉시 반영됩니다.

- **직관적인 기본 단축키**

  - `Ctrl + 1` : 리스트 보기
  - `Ctrl + 2` : 캘린더 보기

- **단축키 커스터마이징 가능**\
  우선순위 설정, 이동, 보기 전환 등을 사용자 취향에 맞게 설정할 수 있습니다.

- **강력한 캘린더 뷰**

  * 모든 프로젝트 + 구독 중인 Google Calendar를 한 화면에서 확인 가능
  * 일정 중심으로 하루·주·월 단위 파악이 매우 수월함

- **부가 기능이 풍부함**\
  아이젠하워 매트릭스, 데스크탑 위젯 등 선택적으로 활용 가능한 기능들이 많습니다.

- **피드백 응답이 빠름**\
  버그 리포트나 기능 요청 시, 최소한 ‘문제 인지 여부’에 대한 답변은 빠르게 받을 수 있습니다.

---

### 단점

- **Windows 앱 한글 입력 문제 (가장 치명적인 단점)**\
  IME 관련 문제로 추정되며, Windows 데스크톱 앱에서 한글 입력이 정상적이지 않습니다. (웹/안드로이드는 문제 없음)
  {{< figure src="img/korean-input-bug-in-ticktick.png" alt="Ticktick 윈도우 프로그램에서 한글 입력이 제대로 안됨" >}}

- **디자인이 다소 과밀함**\
  화면에 정보가 많아 한눈에 들어오긴 하지만, 여백과 색 사용이 아쉬워 조잡하게 느껴질 수 있습니다.\
  (사실 장점과 단점이 섞여있지만 우선 단점에 둠)

## 기타 투두 앱 (간단 정리)

- **마이크로소프트 투두(Microsoft To Do, MS To Do)**\
  캘린더 뷰 없음, 자연어 입력 없음\
  단순한 집안일·개인 할 일 관리 용도로는 충분

- **애니두(Any.do)**\
  한글 입력/검색, UI 등이 매끄럽지 않지만 전반적으로 무난한 수준

- **투두메이트(Todomate)**\
  한글 사용, 한국인들이 사용하기에는 좋지만 자연어 입력 미지원, 캘린더 기능 아쉬움\
  가족 단위 사용에는 적합할 수 있으나, 외부 캘린더 연동이 약함

- **노션(Notion)**\
  일정 관리도 가능하지만, 앱이 무겁고 투두/캘린더 용도로는 과도한 느낌

## 결론

현재 제 기준에서 가장 큰 비교 포인트는 두 가지입니다.

1. 캘린더 사용성 (UI, 사용성, 단축키 등)
2. 한글 UI 및 입력, 사용 등

캘린더 기능만 놓고 보면 TickTick이 압도적으로 우수하지만 실제 사용 비중이 높은 한글 입력과 자연어 처리를 고려하면 TickTick의 한글 입력 문제는 치명적입니다.

결국 한글 UI나 입력 등으로 인해 Todoist를 선택했고, 구독 연장 혹은 계속 사용할 가능성이 높습니다.

물론 Todoist가 캘린더 뷰를 조금만 더 보완한다면, 가격에 대한 고민 없이 바로 선택할 수 있을 것 같습니다.

기능이나 가격만 보면 Ticktick이 가장 좋지만 투두 앱은 결국 나의 작업 흐름과 얼마나 잘 맞는지가 가장 중요한 것 같습니다.
