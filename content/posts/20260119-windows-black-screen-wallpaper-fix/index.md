PC를 사용하다 보면 갑자기 배경화면이 사라지고 검은색 화면만 보이는 경우가 있습니다.

{{< figure src="img/windows-black-screen-error.png" alt="윈도우 바탕화면이 검은색으로 나타나는 오류 화면" >}}

바탕화면만 어둡게 변하고 다른 동작하는 것에 아무런 문제가 없다면 보통 `Windows 탐색기`만 재실행 해주면 문제가 해결됩니다.

원인은 잘 모르겠습니다. 고사양 작업을 할 때나 여러 프로그램들을 돌릴 때, 아니면 특별히 어려운 작업을 한 것도 아닌데도 발생할 때가 있습니다.

오늘은 빠르게 해결하는 방법 알려드리겠습니다.

---

## 1. 윈도우 작업 관리자 접속하기

먼저 시스템 프로세스를 제어하기 위해 작업 관리자를 실행해야 합니다.

* **방법 1**: 단축키 `Ctrl + Shift + Esc`를 동시에 누릅니다.
* **방법 2**: 윈도우 작업 표시줄에서 마우스 오른쪽 클릭 후 `작업 관리자`를 선택하세요.

{{< figure src="img/taskbar-right-click-context-menu.png" alt="작업 표시줄 우클릭 메뉴에서 작업 관리자 항목 확인" >}}

---

## 2. Windows 탐색기 프로세스 재시작

문제 해결을 위해 `Windows 탐색기(explorer.exe)`를 다시 불러오는 과정을 진행하겠습니다.

1. 작업 관리자 상단의 `[프로세스]` 탭을 클릭합니다.
2. 목록에서 `Windows 탐색기`를 찾습니다. (저는 `W`를 눌러 빠르게 찾았습니다.)
3. 해당 항목에서 마우스 오른쪽 클릭을 한 뒤 `[다시 시작(R)]`을 클릭합니다.

{{< figure src="img/task-manager-restart-windows-explorer.png" alt="작업 관리자에서 Windows 탐색기 프로세스 다시 시작 선택" >}}

---

## 3. 배경화면 정상 복구 확인

1. `[다시 시작(R)]`을 클릭하면 작업 표시줄과 아이콘들이 사라졌다가 다시 나타납니다.
2. 잠시 후 사라졌던 배경화면이 정상적으로 다시 보이는지 확인합니다.

{{< figure src="img/windows-desktop-wallpaper-restored.png" alt="탐색기 재시작 후 정상적으로 복구된 윈도우 바탕화면" >}}

{{< alert >}}
다시 시작을 눌렀는데도 배경화면이 나오지 않고 아예 화면이 멈춘 것 같다면, `Win + R`을 누르고 `explorer.exe`를 입력한 뒤 엔터를 누르면 수동으로 탐색기를 실행할 수 있습니다.
{{< /alert >}}

---

## (참고) 명령 프롬프트(CMD)로 해결하기

마우스 조작 없이 명령어 하나로 빠르게 해결하고 싶다면 아래 방법도 가능합니다.\

1. `Win + R`을 눌러 `cmd`를 입력합니다.
2. 아래 명령어를 복사하여 입력한 뒤 실행해봅니다.

```bash
taskkill /f /im explorer.exe & start explorer.exe
```

3. 위 방법과 동일한 과정 및 결과를 얻을 수 있습니다.

---

## 마치며

윈도우 배경화면이 검게 변하는 현상은 대개 시스템 리소스 부족이나 탐색기 프로세스의 충돌로 발생한다고 합니다.

그런데 저는 시스템 리소스도 넉넉하고, 별 프로그램들을 쓰지 않는데도 생각보다 자주 일어나는 것 같습니다.

물론 컴퓨터를 재부팅하면 해결되기도 하지만, PC를 다시 켜면 번거롭고 시간이 오래 걸리기 때문에 제가 알려드린 방법을 쓰시는게 가장 빠르실겁니다.
