챗GPT를 인터넷 브라우저에서 기본 검색 엔진으로 설정하는 방법에 대해 소개해드립니다.

참고로 저는 현재 구글 크롬(Google Chrome)을 메인으로 사용중이기 때문에 크롬 기준으로 설명드리겠습니다.

접근 방법이나 메뉴 구성은 다를 수 있지만 대부분의 메이저 브라우저도 사용방법이 비슷합니다.

**확장 프로그램을 사용하는 방법**과 **브라우저 기본 기능을 활용하는 방법** 이렇게 2가지가 있습니다.

저는 브라우저 기본 기능을 활용하지만 일단 2가지 방법 모두 소개해드리겠습니다.

## 브라우저 확장 프로그램 사용

**장점**: 쉽고 빠르게 주소 표시줄 검색 엔진으로 설정 가능

**단점**: 확장 프로그램에 대한 의존성 발생, 여러 검색엔진을 사용하는 경우 기본 검색엔진이 챗GPT로 고정됨

참고로 저는 확장 프로그램을 사용하지 않지만 쉽고 빠르게 설정할 수 있기에 우선 소개해드립니다.

### 확장 프로그램 설치

1. [Chrome 웹 스토어](https://chromewebstore.google.com/)에 접속한다.
2. 검색창에 **ChatGPT**를 입력한 뒤 [**ChatGPT search**](https://chromewebstore.google.com/detail/chatgpt-search/ejcfepkfckglbgocfkanmcdngdijcgld)를 찾는다.
3. **Chrome에 추가**를 눌러 확장 설치 및 적용한다.

### 확장 프로그램 사용

설치 후 주소 표시줄에 질문을 입력한 뒤 Enter를 눌러 챗GPT 사용이 잘 되는지 확인해봅니다. 특별한 문제가 없다면 바로 검색엔진 활용이 가능하다는 것을 확인할 수 있습니다.이후 챗GPT 로그인 후 계정과 연동하여 활용하면 됩니다.

## 브라우저 기본 검색엔진 활용

저는 확장 프로그램 의존성을 낮추기 위해 기본 기능을 최대한 활용하고 있습니다.

챗GPT 뿐 아니라 퍼플렉시티, 덕덕고 같은 다른 엔진도 함께 알려드리겠습니다.

최근 인공지능(AI)을 활용한 서비스들이 많아지고 있으니 해당 방법이 도움이 많이 될 것 같습니다.

확장 프로그램을 제공하지 않는 서비스도 이용할 수 있는 방법입니다.

그리고 제가 제미나이(Gemini)도 사용중인데, 제미나이는 이런 방식을 응용하는 것이 없어서 아쉽습니다...

### 검색엔진 설정

구글 크롬 기준으로 설명드리겠습니다.

1. **크롬 설정 > 검색엔진** 메뉴에서 브라우저 기본 검색엔진에 대해 설정할 수 있습니다.
2. **검색엔진 및 사이트 검색 관리**에 들어간 뒤, **사이트 검색**에서 `추가`를 클릭한다.
3. 이름과 바로가기, URL을 등록하고자 하는 검색엔진 쿼리 URL에 맞게 설정한다.

- **ChatGPT URL**
    ```text
    https://chatgpt.com/?q=%s
    ```

- **Perplexity URL**
    ```
    https://www.perplexity.ai/search?focus=internet&q=%s
    ```

- **Duckduckgo URL**
    ```
    https://duckduckgo.com/?q=%s
    ```

### 검색엔진 사용

주소 표시줄에 `바로가기` 입력 후 `스페이스바 또는 Tab`을 누르면 입력한 바로가기에 해당하는 검색엔진으로 전환되어 검색가능합니다.

## 제 사용방법

저는 **바로가기**를 활용해 다양한 검색엔진들을 빠르게 활용하고 있습니다.\
제가 활용하는 방법 일부를 아래 예시를 통해 소개해드립니다.

> **기본 검색엔진**: 구글(바로가기: g)\
> **추가 검색엔진**: 챗GPT(바로가기: c), 퍼플렉시티(바로가기: p)

1. 브라우저 사용 중에 **Ctrl + E** 단축키를 눌러 기본 검색 엔진(구글)을 활성화하여 검색한다.
2. 주소 표시줄에 등록한 **바로가기 단축키**를 이용해 검색 엔진을 전환하면서 검색한다.
    - 챗GPT로 검색하고 싶을 때, 주소 표시줄에 `c` 입력 후 띄어쓰기 > 검색어 입력 > Enter
    - 퍼플렉시티로 검색하고 싶은 경우, 주소 표시줄에 `p` 입력 후 띄어쓰기 > 검색어 입력 > Enter

이처럼 바로가기를 어떻게 설정하느냐에 따라 쉽고 따르게 다양한 검색엔진들을 활용할 수 있습니다.
